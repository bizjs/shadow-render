# shadow-render

Render some html in shadow DOM bypass a component, support more frameworks.
提供各个框架的组件来渲染 HTML 内容到 Shadow DOM 中

# 子包

# 如何研发

## 启动研发环境

```bash
# 初始化依赖
pnpm i

# 启用开发
pnpm dev:react
```

## 测试

```bash
# 运行 Unit Test
pnpm test

# 测试覆盖率
pnpm test:cov

# 更新快照
pnpm test -- -- -u
```

## 构建 & 发布

```bash
# 构建
pnpm build
```
